# Test Samples Evaluation

We randomly sampled 1,000 questions from the test set five times independently for manual verification. The numbers of correctly annotated samples in the five runs were 985, 987, 985, 990, and 986, corresponding to an average accuracy of **98.7%**. We believe that the annotation quality is well within an acceptable range.
